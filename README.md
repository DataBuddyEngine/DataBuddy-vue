## 🤔 什么是 DataBuddy 数据兔可视化引擎？
✨ DataBuddy 数据兔可视化引擎是你的开源数据小助手，帮助你快速分析数据并洞察业务趋势，从而实现业务的改进与优化。DataBuddy 拥有丰富的数据源连接，能够通过拖拉拽方式快速制作图表，一键分享让决策更甜蜜～💖
### 👌 DataBuddy 的优势
- ✨超省钱小能手：服务端最低只需一个嵌入式开发板，用内建 Busybox httpd 服务就能轻松搞定部署啦～
- 💖开源小天使：零门槛，线上秒速获取安装，一键升级超省心，所有功能都免费哦～
- 😊点点拖拖超友好：像搭积木一样简单，鼠标轻轻点点拽拽，大屏 UI 设计瞬间完成～
- 🌈全场景小伙伴：多平台安装超灵活，各种嵌入方式都支持，随时随地守护你的需求～
- 🔒安心分享小卫士：多种数据分享方式任你选，安全防护层层把关，分享数据无 worries～
### 📊 DataBuddy 目前可使用的数据源
- 静态数据：Json、CSV、Excel，如果非默认 Json 格式，导入后会自动转换哦，省时省心～
- 数据库：MySQL、MariaDB，其它数据库对接引擎开发中，敬请期待～
- API 数据源：可使用 Restful API 数据作为数据源，支持 GET、POST 请求，支持自定义请求头，支持数据转换，省时省力～
- 实时数据源：支持 WebSocket 协议，实时获取数据，实时可视化，实时更新，更直观～
### 🌈 DataBuddy 功能特性
- 图表类型：支持丰富的图表类型，包括折线图、柱状图、饼图、雷达图、散点图、热力图、地图等等，满足不同场景需求，一键生成图表。
- 图表主题：支持多种图表主题，满足不同场景需求，一键切换。
- 数据筛选：支持数据筛选，满足不同场景需求，快速定位数据。
- 图表分享：支持图表分享，一键生成分享链接，分享给他人，让他人一看就懂。
- 图表导出：支持图表导出，可进行一键打包导出，实现实时制作，实时封装/分享，也可导出为 BDV 文件，实现同系统一键导入。
- 图表数据：支持图表数据，可实时获取数据，实时更新图表。
- 图表分析：支持图表分析，可分析图表趋势，发现问题，提出建议。
